This blog, along with the companion blog at www.martintkrebs.net (not live yet), have been created to realise the start of an idea I had recently...

The Idea.

To develop two identical web applications, in two different web frameworks and deploy them live to two domains, documenting the process along the way in a series of blog posts.

One instance of the app would be written in the Flask/Python web framework and the other in the Seaside/Smalltalk web framework. Each would be deployed on its own domain, martintkrebs.com and martintkrebs.net respectively.

Both would be served from a Linode/Ubuntu server, with Nginx set up to reverse proxy to uWSGI/Gunicorn for the Flask app, and to the Seaside app server for the Seaside app.

The Start.

As I needed a blogging platform for project documentation, It seemed like a good idea to make the web app itself a blog, so the app could be used to document its own development!

Development will proceed in increments and this is the first (this blog page) - a simple 'Hello World' type app. Functionality will be added in each increment and a blog post created for documentation.
